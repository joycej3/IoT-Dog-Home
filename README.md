## Getting started
Install the Espressif IDF extension (Could take a while)
To build the hello world program
    -ctrl + shift + p
    -Open ESP-idf terminal
    -cd into correct folder
    -type "esp.py build" into terminal
    -to flash type: "idf.py flash"
